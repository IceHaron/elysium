Начать играть


1. Внимательно прочитайте правила и описание серверов.

 

2. Зарегистрируйтесь на нашем сайте под ником, который Вы хотите использовать для игры на сервере.

 

3. Скачайте наш лаунчер для игры. Устанавливается он автоматически в отдельную папку.

 

4. Запустив лаунчер, введите Ваши логин и пароль, которые Вы использовали для регистрации на форуме.

 

5. Выберите сервер из списка доступных и нажмите "войти".

 

6. Приятной Вам игры и увлекательных приключений на просторах серверов Elysium Game.
